# VIZZU Digital - Landing Page

Landing page profissional para servicos de customizacao de lojas FacilZap.

## Arquivos

- `index.html` - Landing page principal
- `admin.html` - Painel de gestao de leads
- `netlify.toml` - Configuracao do Netlify

## Deploy

### Netlify (Recomendado)

1. Acesse [netlify.com](https://netlify.com)
2. Clique em "Add new site" > "Import an existing project"
3. Conecte sua conta GitHub
4. Selecione o repositorio `VIZZUDIGITAL`
5. Clique em "Deploy site"

### GitHub Pages

1. Va em Settings > Pages
2. Source: Deploy from a branch
3. Branch: main / root
4. Save
